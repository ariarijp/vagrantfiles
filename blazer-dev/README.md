# blazer-dev

Vagrantfile for [ankane/blazer-dev](https://github.com/ankane/blazer-dev).

## Usage

First, Run below commands.

```bash
$ vagrant up
$ vagrant ssh
$ cd /opt/blazer-dev
$ ./bin/rails s -b 0.0.0.0`
```

Well done! you can access to Blazer on [http://localhost:3000](http://localhost:3000).

Enjoy!
